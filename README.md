## 🚀 프로젝트 개요 (Project Overview)
**ROSAE HUB**는 단순한 채팅 앱을 넘어, **블로그, 실시간 커뮤니티, 그리고 디지털 오픈 마켓**이 결합된 차세대 웹 플랫폼입니다.  
사용자 경험(UX)과 감성적인 디자인(Emotional Tech)에 초점을 맞추었으며, 최신 웹 기술(PWA)을 통해 앱과 같은 부드러운 사용성을 제공합니다.

---

## 🌟 주요 기능 (Key Features)

### 1️⃣ 기술 블로그 & SEO 최적화 (Tech Blog)
*   **검색 엔진 최적화(SEO)**: 구글 검색 시 최상단에 노출되도록 메타 태그, sitemap.xml, 시멘틱 마크업이 완벽하게 적용되었습니다.
*   **콘텐츠 관리**: 마크다운 기반의 기술 포스팅과 세련된 읽기 환경을 제공합니다.

### 2️⃣ 실시간 채팅 & 커뮤니티 (Real-time Chat)
*   **채널 시스템**: 대규모 인원이 참여하는 공개 채널부터, 비밀번호가 걸린 비공개 채널까지 지원합니다.
*   **친구 & 초대**: 친구 추가, 차단, 그리고 채널 초대 기능을 통해 프라이빗한 소통이 가능합니다.
*   **반응형 UI**: 모바일과 데스크톱 어디서든 완벽한 비율로 동작하는 반응형 디자인을 갖췄습니다.
*   **이모티콘 시스템**: 텍스트뿐만 아니라 다양한 전용 이모티콘으로 풍부한 감정 표현이 가능합니다.

### 3️⃣ 이모티콘 오픈 마켓 (Open Market Economy) [NEW] ✨
*   **가상 화폐 시스템(Coin)**: 사용자는 코인을 충전하거나 보상으로 획득하여 지갑(Wallet)에 보유합니다.
*   **디지털 굿즈 상점**: 상점(Shop)에서 유니크한 이모티콘 팩을 구매할 수 있습니다.
*   **소유권 및 사용**: 구매한 아이템은 즉시 내 보관함에 연동되어 채팅에서 사용할 수 있습니다.
*   **확장성**: 추후 크리에이터가 직접 이모티콘을 등록하고 수익을 창출할 수 있는 생태계를 지향합니다.

## 🛠 기술 스택 (Tech Stack)
*   **Frontend**: HTML5, CSS3 (Modern Variables), Vanilla JavaScript (ES6+)
*   **Backend**: Supabase (PostgreSQL, Authentication, Realtime DB)
*   **Deployment**: Cloudflare Pages / GitHub Pages

## 💎 비즈니스 모델 & 로드맵
우리는 **"감성 테크(Emotional Tech)"**를 지향합니다. 딱딱한 기계적 인터페이스가 아닌, 따뜻하고 몰입감 있는 디지털 안식처를 만드는 것이 목표입니다.
*   **현재**: 채팅 및 기본 마켓 기능 안정화
*   **미래**: 크리에이터 수익구조(Creator Economy), 음성/화상 채팅, 나만의 테마 커스터마이징
